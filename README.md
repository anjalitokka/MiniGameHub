# 🎮 Mini Game Hub

A modern browser-based gaming platform that brings multiple classic games together in one application. Built using **HTML, CSS, and JavaScript**, the project features an attractive landing page, smooth navigation, responsive design, and interactive gameplay.

## 📌 Features

- 🎮 Interactive Mini Game Hub
- 🏠 Modern home dashboard
- ✨ Glassmorphism UI with animations
- 📱 Fully responsive design
- 🎨 Smooth hover effects and transitions
- ℹ️ About popup
- 🔙 Easy navigation between games
- 🚀 Expandable architecture for adding more games
